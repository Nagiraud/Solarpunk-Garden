# The Drone Garden

Avec votre drone, plantez des graines, arrosez-les pour les faire grandir, puis récoltez-les avant qu'elles n'explosent  ! 

Si vous atteignez 1000 points, c'est gagné !

Mais si vous ne ramassez pas assez vite vos fruits, il risque d'exploser, et c'est la défaite assurée !



Ce jeu a été réalisé en 3ème année de BUT informatique au Puy-en-Velay
